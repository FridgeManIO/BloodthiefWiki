Install the python dependencies from requirements.txt, or follow the guide on [material mkdocs](https://squidfunk.github.io/mkdocs-material/getting-started/)

To build the site from source, run `mkdocs serve` for a liveserver, or `mkdocs build` for a one-time build.

Ensure the link at the top of `mkdocs.yml` points to your hosted server.

Deploy the contents generated inside the "Site" directory to your chosen host.
